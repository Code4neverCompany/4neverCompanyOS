# Install — macOS

## System Requirements

- macOS 12 Monterey or later (Intel or Apple Silicon)
- ~2 GB disk space
- Internet connection for initial setup
- Anthropic API key (for Dev persona / Claude Code)
- Google account (for Antigravity CLI — optional, needed for Frontend Designer)

## Download

Download the latest `.dmg` installer from the [releases page](https://github.com/Code4neverCompany/4neverCompanyOS/releases).

## Installation

1. Open the downloaded `.dmg` file
2. Drag **4neverCompany OS** into the **Applications** folder
3. On first launch, macOS may show a gatekeeper prompt. Click **Open** anyway — the app is signed once Apple Developer ID is in place
4. The workspace launches and the first-run wizard appears

## Build from Source

If you are building from source, see [Build from Source](/install/build).

## Next Steps

After installing, run the [First Run Wizard](/first-run/) to configure credentials and your vault location.
